# Variant: Shadow Deployment
**New model runs alongside production on real traffic but outputs are discarded. Zero-risk validation.**
## Live Demo
**[Launch](https://aman210122.github.io/ai-architecture-enterprise-patterns/patterns/05-llmops-pipeline/shadow-deployment/index.html)**
**Use:** Validate new model on real production traffic without any user impact. Compare quality, latency, cost side-by-side. Different from A/B testing (which serves real responses to users).
