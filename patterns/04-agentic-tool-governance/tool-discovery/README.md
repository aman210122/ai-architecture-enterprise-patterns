# Variant: Tool Discovery & Marketplace

**Agents dynamically discover tools at runtime from an internal marketplace. Search, evaluate, approve, invoke.**

## Live Demo
**[Launch](https://aman210122.github.io/ai-architecture-enterprise-patterns/patterns/04-agentic-tool-governance/tool-discovery/index.html)**

Scenarios: Known tool call, unapproved tool blocked, new tool approved on demand, search + discover + invoke, discovery timeout.

**Use:** Large enterprise with 200+ internal APIs. Agents cannot have all tools pre-loaded. Runtime tool discovery with governance approval. Internal tool marketplace.
