# Variant: Output Quarantine

**Suspected outputs held in quarantine queue for human review. Clean outputs flow immediately.**

## Live Demo
**[Launch](https://aman210122.github.io/ai-architecture-enterprise-patterns/patterns/07-contamination-resistant-pipeline/output-quarantine/index.html)**

**Use:** When you cannot afford to block all suspected outputs (too many false positives) but also cannot deliver unchecked. Human review queue for borderline cases.
